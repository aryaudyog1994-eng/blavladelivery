
# BlaVlaDelivery - Copyright-safe Frontend Scaffold

This package is a mobile-first frontend scaffold for a parcel delivery app inspired by ride-sharing apps.
Design is original and copyright-safe (colors, assets and text are unique).

Files included:
- index.html (home)
- splash.html
- login.html, signup.html
- user-kyc.html, driver-kyc.html
- add-parcel.html, parcel-tracking.html
- chat.html, admin.html
- agreement.html, about.html, contact.html, how-it-works.html
- assets/styles.css, assets/js/main.js, assets/images/logo_blue.jpg, assets/images/logo_grey.jpg

How to deploy:
1. Extract files to root of GitHub repository.
2. Commit & push; enable GitHub Pages (branch: main).
3. For real features, integrate Firebase and Maps (see assets/js/main.js placeholders).
